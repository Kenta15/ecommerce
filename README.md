## Setup
```
# Download Python
https://www.python.org/downloads/

# Download Anaconda
https://www.anaconda.com/products/distribution

# Install Django
python -m pip install Django

# Run these commands
conda activate djangoenv
cd ecommerce
python manage.py runserver
```
